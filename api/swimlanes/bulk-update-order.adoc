To update the order of multiple swimlanes at the same time send a POST request with the following data:

- *project* (required)
- *bulk_swimlanes*: list where each element is a list, the first element is the swimlane id and the second the new order

include::../generated/swimlanes-bulk-update-order-cmd.adoc[]

When the update is successful, the HTTP response is a 204 NO CONTENT with an empty body response
