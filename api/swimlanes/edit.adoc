To edit swimlanes send a PUT or a PATCH specifying the swimlane id in the url.
In a PATCH request you just need to send the modified data, in a PUT one the whole object must be sent.

include::../generated/swimlanes-patch-cmd.adoc[]

When the creation is successful, the HTTP response is a 200 OK and the response body is a JSON link:#object-swimlane-detail[swimlane detail object]
